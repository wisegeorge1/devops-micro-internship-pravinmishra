# 🚀 GitHub Account Setup Guide (Beginner Friendly)

This is a complete step-by-step guide to create and set up your GitHub account.

If you **don’t already have a GitHub account**, follow this guide carefully and complete all steps.

If you **already have a GitHub account**, you can skip this file and move on to the next onboarding guides in the repository.

---

1. Sign in to your gmail account  

---

2. Copy paste this link in your browser  
   [https://github.com/signup](https://github.com/signup)

---

3. You’ll navigate to this page:  
![Step 3](images/ss1.png)

---

4. Provide your email  
![Step 4](images/ss2.png)

---

5. Provide a strong password for your account - Consider the password policies highlighted. Otherwise it will throw you a warning  
![Step 5](images/ss3.png)

---

6. Add your username according to this pattern  

DMI-C3-YOURFIRSTNAME } YOURFIRSTNAME=your first name  

![Step 6](images/ss4.png)

---

7. Add your country, Check the box to signup for copilot.  
![Step 7](images/ss5.png)

---

8. Select ‘Create account’  
![Step 8](images/ss6.png)

---

9. To verify your account , choose ‘Visual puzzle’  
![Step 9](images/ss7.png)

---

10. Solve the simple puzzle and select ‘Submit’ after solving - you’ll get random puzzle, not the same one as this  
![Step 10](images/ss8.png)

---

11. After successfully solving the puzzle, you’ll get a window like this  
![Step 11](images/ss9.png)

---

12. Go to your email inbox (The one you provided when you creating this account)   
    You’ll see an email with the github launch code as shown below.   
    Copy that code.  
![Step 12](images/ss10.png)

---

13. Come to the github account creation window again and paste your code here.  

It will verify the code and automatically continue to the next window.  
![Step 13](images/ss11.png)

---

14. If you see a window like below, Congratulations! YOu have created your github account successfully.  
![Step 14](images/ss12.png)

---

15. Sign in to your github account using the username and password you given when creating the account.  

Provide your username, password & Click ‘Sign in’  
![Step 15](images/ss13.png)

---

16. Voila! You’re inside your created github account now!  
You need to see an output window like this.  
![Step 14](images/ss14.png)

---

17. Update your details in the profile  
Click on the icon highlighted below.  
![Step 17](images/ss15.png)

---

18. Click on profile  
![Step 18](images/ss16.png)

---

19. Select ‘Edit profile’  
![Step 19](images/ss17.png)

---

20. Provide your name - This is your actual name, not the github username  
![Step 20](images/ss18.png)

---

21. Add your bio  
![Step 21](images/ss19.png)

---

22. Set the pronoun  

Click the icon highlighted below - you’ll get a dropdown menu with options  
Select the pronoun  
![Step 22](images/ss20.png)

---

23. Add your company & your country  
![Step 23](images/ss21.png)

---

24. Click the checkbox ‘Display current local time’  
Select the icon mentioned - You will get dropdown menu with time zones  
![Step 24](images/ss22.png)

---

25. Select your correct time zone among them  
![Step 25](images/ss23.png)

---

26. Add your email, Add your website link if you have any, otherwise you can leave it.  
![Step 26](images/ss24.png)

---

27. Add your professional social profiles like linkedin, medium or any blog post site etc  
Then click ‘Save’  
![Step 27](images/ss25.png)

---

28. You can see your details updated like this:  
![Step 28](images/ss26.png)

---

29. Click on this frame  
![Step 29](images/ss27.png)

---

30. Select edit icon for the profile picture  
![Step 30](images/ss28.png)

---

31. Click on ‘Upload a photo’  
![Step 31](images/ss29.png)

---

32. Select one of your professional photographs from your local machine and click ‘Open’  
![Step 32](images/ss30.png)

---

33. Adjust your photo to the circle shaped frame  

Click on ‘Set new profile picture’  
![Step 33](images/ss31.png)

---

34. You can see your github profile picture updated in these place  

P.S: You may not see the profile picture updated right after you upload it. In such cases, refresh a couple of times. If it is still not displaying, You can close the tab and open it again.  

![Step 34](images/ss32.png)

---

# 🎯 You’re Done!

If you followed all the steps, you now have:

- A GitHub account created  
- Your email confirmed (you checked your inbox and entered the code GitHub sent you)  
- A completed profile  

👉 If you already finished this, move on to the next onboarding file in this folder and continue from there.
